Image Modal

An image modal is an example of a modal box.
We are able to click on the image and have it fill a box that appears on the page,
but remains on the current page.
This means a shorter load time, since we are not going to a new page

Use a JavaScript function to trigger the modal and to display the current image inside the modal when it is clicked on. The value from the image's "alt" attribute becomes the image's caption.
